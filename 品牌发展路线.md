## 五个阶段

### 阶段一 · 筑基（0–6 个月）→ 品牌资产 + 网站

**目标**：AndDream 的视觉和数字存在完整到可以"拿出来给人看"。

| 里程碑 | 具体产出 | 优先级 |
|--------|---------|:---:|
| **Logo 定稿** | 基于现有 engraving 风格，产出一版可矢量使用的 logo 标识 | 🔴 |
| **品牌规范书** | 一页纸 PDF/网页：色板（浅+深）、字体、装饰元素使用规则、禁例 | 🔴 |
| **网站 v1 上线** | GitHub Pages 部署，浅/深双模，作品展示区 | 🔴 |
| **PPT 母版** | 一套 16:9 雕刻风格 PPT 模板（封面/目录/正文/结尾） | 🟡 |
| **社交统一视觉** | 小红书、公众号、GitHub 头像+封面 | 🟡 |
| **生成艺术封面** | p5.js 参数化品牌封面，可交互 | 🟡 |

**检验标准**：有人问"你品牌长什么样"，你能一个链接发过去。

---

### 阶段二 · 发声（6–12 个月）→ 内容输出

**目标**：让 AndDream 不只是"一个好看的页面"，而是"一个有人在说话的品牌"。

| 渠道 | 内容方向 | 频率 |
|------|---------|:---:|
| **小红书** | 设计过程拆解、AI 工具技巧、视觉灵感 | 周更 1–2 篇 |
| **公众号/博客** | 设计哲学长文、品牌复盘、工具深度评测 | 月更 1–2 篇 |
| **GitHub** | 开源 engraving-brand skill、PPT 模板 | 有产出即更新 |

**内容策略**：
- 不追热点——追热点稀释品牌识别度。写你真正有见解的东西
- 每篇内容都是一次品牌接触——配图、文案语气、视觉风格统一用 AndDream 规范
- 把过程变成内容——做 PPT、调动画、改配色本身就是素材

**检验标准**：小红书 500 粉，有一篇点赞过 500。

---

### 阶段三 · 沉淀（12–18 个月）→ 建立作品壁垒

**目标**：积累不可替代的作品资产，让品牌在细分领域被认出来。

| 方向 | 产出 |
|------|------|
| **作品集网站 v2** | 10+ 个精选项目，每个有完整的 case study（过程+决策+结果） |
| **设计系统开源** | Engraving Style Guide 完整发布，含 Figma/代码/文档 |
| **系列化内容** | 一个固定栏目（如"用 AI 做设计"每周一期），形成 IP |
| **跨平台联动** | 小红书引流→公众号深度内容→GitHub 下载→Product Hunt 发布 |

**检验标准**：有人因为看到你的作品主动联系合作，月均 1–2 次。

---

### 阶段四 · 产品化（18–24 个月）→ 工具/Skill

**目标**：从"自己用的工具"升级为"别人也能用的产品"，产生第一笔收入。

| 产品 | 形态 | 盈利模式 |
|------|------|---------|
| **Engraving Skill** | Claude Code Skill（开源） | 免费 → 建立声誉 |
| **雕刻风格 PPT 模板包** | 10+ 页模板 .pptx | 付费下载（¥29–59） |
| **更多风格 Skill** | 开发 2–3 个不同风格的 Claude Code Skill | 免费+付费混合 |
| **品牌设计轻咨询** | 1v1 帮人做品牌视觉 | 按项目收费 |

**检验标准**：有一个付费产品上线，月收入覆盖一顿饭。

---

### 阶段五 · 工作室化（24–36 个月）→ 可持续接单

**目标**：AndDream 成为可持续运营的创意品牌，有稳定客户和收入。

| 服务类型 | 说明 |
|---------|------|
| **PPT 定制** | 企业/学术/路演 PPT 设计，雕刻风格或其他定制 |
| **品牌视觉** | Logo + 色板 + 字体 + 网页全套 |
| **AI 创意咨询** | 帮客户用 AI 做设计、搭工作流 |
| **模板/Skill 持续收入** | 维护已有产品，开发新品，建立被动收入 |

**检验标准**：月均接到 2–3 单付费委托，品牌能自负盈亏。

---

## 关键决策点

| 时间 | 评估问题 |
|------|---------|
| **6 个月** | 品牌资产是否完成？网站有没有人访问？ |
| **12 个月** | 内容是否产生互动？有没有人主动联系？ |
| **18 个月** | 作品壁垒是否形成？有没有复利效应？ |
| **24 个月** | 产品有没有人付费？要不要加大投入？ |
| **36 个月** | 工作室模式是否可持续？要不要扩大/转型/全职？ |

---

## 不做什么

- ❌ 不做纯技术 SaaS——这不是你的差异化优势
- ❌ 不追 AI 热点做内容——"GPT-X 来了！"这种内容稀释品牌
- ❌ 不接低价走量单——¥50 做一套 PPT 会拉低品牌定位
- ❌ 不着急注册公司——等有稳定收入再考虑法律实体
- ❌ 不把品牌搞得太像"创业公司"——保持个人品牌的手工感和真诚感
