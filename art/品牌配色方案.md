# AndDream 品牌配色方案

> 设计决策备忘录 · 2026/6/19

---

## 前提

AndDream 的视觉锚点是**古典铜版雕刻**——这是已确立的方向，不能丢。配色的讨论边界是：在 B&W 雕刻基底之上，是否引入色彩，引入什么色彩。

---

## 判据

| 维度 | 要求 |
|------|------|
| 不能稀释雕刻感 | 任何色彩必须能被理解为"油墨的延伸"，而非"UI 装饰" |
| 承载品牌语义 | 色彩必须和 "And..."（延续）、"Dream"（梦境）有内在关联 |
| 区别于 AI 默认审美 | 避开 cream+terracotta / dark+acid-green / broadsheet 三大模板 |
| 可跨媒介 | 网页、PPT、生成艺术、印刷品统一使用 |

---

## 三个方向

### 方向一 · 纯雕刻（Pure Engraving）

**严格 B&W，不加任何色彩。**

墨色 `#111111` → 纸色 `#FAFAF8`。唯一的"颜色"是透明度——用 opacity 制造灰色层次。hover 反馈是墨色加深、letter-spacing 展开，而不是变色。

| 色板 | |
|------|------|
| 纸色 | `#FAFAF8` |
| 墨色 | `#111111` |
| 浅墨 | `#3A3A3A` |
| 刻线 | `#1A1A1A` |

**判断**：最纯粹、最忠于铜版画原作。风险是太冷——"Dream"有温度，纯黑白可能缺一点暖意。适合如果品牌就是走"极简克制"路线。

---

### 方向二 · 靛蓝页缘（Indigo Margin）⭐ 推荐

**B&W 为基底，引入一个极度克制的深靛蓝作为唯一强调色。**

18 世纪版画里，**普鲁士蓝**是当时最革命性的合成颜料——生于 1704 年，恰好与铜版雕刻的黄金时代重合。把这一抹蓝用在链接、hover 状态、仅有的一个装饰元素上，其余一切保持 B&W。

| 角色 | 色值 | 用途 |
|------|------|------|
| 纸色 | `#FAFAF8` | 背景 |
| 墨色 | `#111111` | 主体文字 |
| 浅墨 | `#3A3A3A` | 次级文字、中文段落 |
| 刻线 | `#1A1A1A` | 装饰线、边框 |
| **靛蓝** | `#254E7A` | **强调色**：hover、链接、仅 1-2 个装饰元素 |
| 蓝灰 | `#4A78B0` | 靛蓝的浅色变体（极少使用） |

**设计意图**：
- 靛蓝在 `#1A2744` 这个深度上——乍看近乎黑，只在 hover 或光线下"显蓝"
- 语义契合：蓝色 = 夜空、梦境、潜意识、"未完"的深邃
- 风险：用户可能看不出用了颜色。**这正是风险本身**——一个需要第二眼才能发现的色彩系统，本身就是"雕刻感"的延伸

**使用规则**：
- 每页/每屏最多 **1-2 处**出现靛蓝
- 不做蓝色背景块，不做蓝渐变
- hover 时墨色 → 靛蓝（`#111 → #254E7A`），transition 0.5s+
- 实际落地时 #1A2744 在屏幕上近乎黑色不可辨识，上提到 #254E7A 才有可见蓝调

---

### 方向三 · 铜版余温（Warm Plate）

**B&W 基底 + 暖赭石色点缀。**

铜版画印刷时，油墨在纸上微微渗透、氧化后的暖调。赭石色来自铁氧化物——最古老的颜料之一，和版画同时代。

| 角色 | 色值 | 用途 |
|------|------|------|
| 纸色 | `#FAF8F3` | 比纯雕刻微暖一度 |
| 墨色 | `#141110` | 带极微暖调的墨 |
| 浅墨 | `#3D3834` | 次级文字 |
| 刻线 | `#1C1917` | 装饰线 |
| **赭石** | `#8B5A3C` | 强调色 |
| 暖金 | `#B8936E` | 赭石的浅色变体 |

**判断**：温暖、有人情味。但 "cream + terracotta accent" 是 AI 生成设计的三大默认审美之一——即使赭石 ≠ terracotta，仍有撞车风险。适合如果品牌需要更多"人的温度"。

---

## 推荐：方向二 · 靛蓝页缘

理由：
1. **历史锚点**：普鲁士蓝诞生于 1704 年，和铜版雕刻共享时代背景——不是强加的"UI 色"，而是从版画世界里自然生长出来的
2. **语义契合**：梦境→夜空→深蓝。"And..." 的未完感→蓝色的深度和神秘
3. **最不 AI**：B&W + 极度克制的深蓝是三个方向里最不容易撞 AI 默认审美的
4. **可升级**：如果未来需要更多色彩（比如产品 UI），可以从靛蓝延伸出一套冷调单色系统，而不破坏 B&W 基底

---

## 实施建议

### 第一步：雕刻基底先做扎实

纯 B&W 版本要保持为"默认态"。靛蓝只在有交互或需要引导注意力的地方出場。

### 第二步：仅改 hover 色

把当前所有 `:hover` / `:active` 的颜色变化目标从 `#000 / #222` 改为 `#1A2744`。这是最安全的第一步——用户可能都没注意到变了色，但整体"气质"会更沉静。

### 第三步：选一个元素做"蓝色宣言"

在整个品牌页里，**只选一个元素**用靛蓝——比如封面的大 Flourish 中心那颗菱形，或者 "And..." 的省略号，或者 `anddream-brand/index.html` 里 `.pull` 引文的上下边框。一处的蓝让整页有了"颜色记忆点"。

### 第四步：生成艺术调色

`engraved-cover.html` 的线条 ink alpha 里混入微量的蓝 bias——R 通道略低于 G/B，在纯黑中埋入极微的冷调。肉眼不一定分辨得出，但和纯 #000 排线并排对比时，冷调版本会"更安静"。
